## 🚀 Highlight Project

### 🔒 Rewire — Automated Grading System

A distributed platform designed to replace traditional manual evaluation with a **fully automated, objective grading pipeline**.

**Core capabilities:**

* Repository analysis (architecture + functionality)
* Containerized execution (Docker-based runners)
* Distributed workers (Go + Redis Pub/Sub)
* Parallel problem evaluation
* Skill-based grading system
* AI-generated feedback (weaknesses, improvements, motivation)

**Architecture:**

* Laravel (Octane + FrankenPHP)
* Nuxt.js
* PostgreSQL (AWS RDS)
* Redis (ElastiCache)
* Docker + Go workers
* AWS EC2 infrastructure
